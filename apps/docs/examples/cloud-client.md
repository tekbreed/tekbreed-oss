# Cloud client example

Use `@tekmemo/cloud-client` as the only package that talks directly to TekMemo Cloud.

Source folder:

```txt
examples/cloud-client/
```

Run it from the repository root:

```bash
pnpm --filter @tekmemo/example-cloud-client dev
```

Validate it:

```bash
pnpm --filter @tekmemo/example-cloud-client typecheck
```
