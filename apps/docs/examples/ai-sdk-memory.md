# AI SDK Memory

Use `@tekmemo/ai-sdk` when an AI SDK application needs memory-aware tools or prompt context helpers.

```sh
pnpm add tekmemo @tekmemo/fs @tekmemo/ai-sdk
```

Start with local memory in development, then add recall providers when the app needs semantic retrieval.
