# Recall with Upstash

Use this path when your TypeScript app needs hosted vector search while keeping TekMemo records provider-neutral.

```sh
pnpm add @tekmemo/recall @tekmemo/upstash-vector @tekmemo/openai
```

Use `@tekmemo/voyageai` instead of `@tekmemo/openai` if Voyage is your embedding provider.
