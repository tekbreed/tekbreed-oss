# TypeScript SDK

The TypeScript SDK is the current TekMemo developer surface.

Start with the core runtime and local filesystem adapter, then add AI SDK, recall, embedding, vector, or reranking packages as your application needs them.

## Packages

| Package | Role |
| :--- | :--- |
| `tekmemo` | Core memory model and runtime contracts. |
| `@tekmemo/fs` | Local filesystem persistence. |
| `@tekmemo/ai-sdk` | Vercel AI SDK integration. |
| `@tekmemo/recall` | Semantic recall orchestration. |
| `@tekmemo/openai` | OpenAI embedding adapter. |
| `@tekmemo/voyageai` | Voyage AI embedding adapter. |
| `@tekmemo/upstash-vector` | Upstash Vector adapter. |
| `@tekmemo/rerank` | Reranking contracts. |
| `@tekmemo/rerank-voyage` | Voyage AI reranking adapter. |
