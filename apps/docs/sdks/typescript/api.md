# API

The TypeScript API is split by package boundary.

| API area | Docs |
| :--- | :--- |
| Core runtime | [Runtime API](/api/runtime) |
| Package imports | [Package API](/api/packages) |
| Configuration | [Config API](/api/config) |
| Package-specific docs | [Packages](/packages/) |

Public package exports should be imported from their package root unless a package page documents a specific subpath.
