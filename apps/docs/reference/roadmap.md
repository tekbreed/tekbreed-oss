# Roadmap

This roadmap is public-facing. It explains where TekMemo is going without exposing internal execution details.

## v0.1: local memory foundation

- `.tekmemo/` filesystem
- core memory
- notes
- conversation JSONL
- event log
- local filesystem adapter
- AI SDK tool integration
- local keyword recall

## v0.2: semantic recall

- chunk registry
- OpenAI embedder
- Voyage embedder
- Upstash Vector adapter
- BYO-provider examples
- source re-indexing
- stale chunk handling

## v0.3: cloud beta

- hosted projects
- API keys
- hosted memory APIs
- hosted deployment guides
- restore points
- usage dashboard
- billing plans
- team access

## v0.4: trust and inspection

- memory inspector
- event timeline
- conflict detection
- memory diff
- manual forget/restore
- usage alerts
- audit exports

## v0.5: ecosystem

- MCP server
- CLI
- connector SDK
- GitHub connector
- local folder connector
- repo-aware memory

## v0.6 and beyond

- graph memory
- temporal recall
- memory decay
- benchmark suite
- advanced reranking
- enterprise governance
- SSO and audit controls
