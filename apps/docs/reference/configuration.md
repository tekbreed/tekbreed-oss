# Configuration reference

| Name | Used by | Purpose |
| --- | --- | --- |
| `TEKMEMO_CLOUD_URL` | Cloud client, CLI, MCP | Cloud API base URL ending in `/api/v1`. |
| `TEKMEMO_API_KEY` | Cloud client, CLI, MCP | TekMemo Cloud API key. |
| `TEKMEMO_PROJECT_ID` | Cloud client, CLI, MCP | Default Cloud project ID. |
| `TEKMEMO_WORKSPACE_ID` | CLI/MCP caller config | Optional workspace context. |
| `TEKMEMO_RUNTIME` | CLI/MCP | Runtime mode. |
