# Competitor Positioning

TekMemo should not compete as just another memory API.

## Positioning

> TekMemo is the developer-owned memory filesystem and runtime for AI apps and agents.

## Differentiators

| TekMemo advantage | Why it matters |
| :--- | :--- |
| File-first | Developers can inspect, diff, and version memory. |
| Local-first | Testing starts without cloud cost. |
| TypeScript-first | Fits modern JS/TS app stacks. |
| Cloud-syncable | Production teams can upgrade without changing the model. |
| Event log | Memory changes become traceable. |
| Future graph memory | Relationships become first-class. |
