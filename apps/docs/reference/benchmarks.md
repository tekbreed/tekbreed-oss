# Benchmarks

TekMemo should avoid unsupported claims until benchmarks exist.

## Benchmark categories

- Recall accuracy
- Temporal recall
- Conflict handling
- Retrieval latency
- Token savings
- Local indexing speed
- Cloud sync reliability
- Storage overhead
- TypeScript integration time

## Public benchmark principle

Every benchmark should include the dataset shape, configuration, provider, vector dimensions, and reproducible instructions.
