# Glossary

## Agent memory

Durable information an AI agent can use across turns, sessions, or workflows.

## Core memory

Small, canonical memory that is usually safe to include in model context.

## Archival memory

Longer notes, summaries, decisions, and conversation records that are retrieved when relevant.

## Recall

The process of finding useful memory for a task.

## Memory compiler

The component that turns raw memory and recall hits into prompt-ready context.

## Chunk registry

A record of which chunks came from which source memory file.

## Event log

Append-only history of memory changes.

## Graph memory

Entities and relationships extracted from memory sources.

## BYO provider

A setup where the developer uses their own embedding provider or vector database instead of TekMemo-hosted infrastructure.
