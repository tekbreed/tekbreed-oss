# FAQs

## What is TekMemo?

TekMemo is a file-first memory runtime for AI apps and agents.

## Can I test TekMemo for free?

Yes. Use `tekmemo` with `@tekmemo/fs` for local file memory. Local keyword recall also costs nothing.

## Do I need hosted TekMemo infrastructure?

No. The current TypeScript SDK runs inside your app. You can start locally with files and add provider-backed recall only when your product needs it.

## What makes TekMemo different?

TekMemo starts with inspectable `.tekmemo/` files. That makes memory easier to debug, version, move, and trust.

## Is the API versioned?

Yes. The latest API docs live at `/api/`, while previous versions live under routes like `/api/v0.1/`.
