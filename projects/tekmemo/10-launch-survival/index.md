# 1-Month Survival Launch Control Center

## Mission

Ship **TekMemo OSS live** and **TekMemo Cloud beta live** within one month, with a visible path to first income.

This is not the ideal long-term roadmap. This is the survival launch path.

---

# Success by end of month

## OSS

- public repo live
- docs live
- core package line usable
- examples available
- package map understandable

## Cloud beta

- public landing/pricing page
- signup or beta intake
- dashboard usable
- project creation works
- core memory works
- notes work
- recall works basically
- API keys work
- usage visible

## Revenue

At least one active path:

- paid beta
- setup package
- architecture review
- direct integration help
- founding-user offer

# Daily rule

Every day, ask:

```txt
Does this help OSS live, Cloud beta live, onboarding, pricing, API usage, or revenue?
```

If not, postpone it.
