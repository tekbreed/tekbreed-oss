# Advanced Packages

These packages should not block the first launch. They are roadmap packages.

## `@tekbreed/tekmemo-graph`

Graph memory contracts and graph expansion.

Use later for:

- relationship memory
- dependency memory
- entity graphs
- project knowledge graphs

## `@tekbreed/tekmemo-connectors`

Connector framework for importing external sources.

Use later for:

- docs ingestion
- GitHub ingestion
- Slack/Discord-like sources
- file source manifests

## `@tekbreed/tekmemo-mcp-server`

MCP server exposing TekMemo tools.

Use later for:

- coding agents
- Claude/OpenAI-compatible tool environments
- local memory server workflows

## `@tekbreed/tekmemo-cli`

Developer CLI for:

- init
- inspect
- index
- search
- benchmark
- doctor

## `@tekbreed/tekmemo-cloud-sync`

Client-side sync contracts for TekMemo Cloud.

It should not include the closed-source cloud backend.

## `@tekbreed/tekmemo-evals`

Recall and memory quality evaluation tools.

## `@tekbreed/tekmemo-benchmark-kit`

Benchmark harness for:

- retrieval quality
- reranking quality
- cost
- latency
- provider comparison

## `@tekbreed/tekmemo-observability`

Telemetry/event hooks for package users.
