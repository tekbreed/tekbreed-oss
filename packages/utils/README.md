# @repo/utils

Internal utility helpers for the **TekMemo** monorepo.

> [!IMPORTANT]
> This is an internal-only package. It is not intended for public use or publishing.

## Contents

- **Common Helpers**: Shared logic used by multiple packages.
- **Internal Constants**: Repo-wide configuration values.
- **Type Utilities**: Shared TypeScript helper types.

## Usage

Reference in `package.json`:

```json
"devDependencies": {
  "@repo/utils": "workspace:*"
}
```
