# @tekbreed/tekmemo-mcp-server

## 1.0.0-alpha.0

### Major Changes

- Initial release of `@tekbreed/tekmemo-mcp-server` — Model Context Protocol server for TekMemo.

  - Exposes TekMemo memory operations as MCP tools for LLM agents
  - `tekmemo-mcp` binary — drop-in stdio MCP server
  - Configurable runtime mode (`local`, `memory`, `cloud`, `hybrid`)
  - `@modelcontextprotocol/sdk` included as a direct dependency — no separate install required

### Patch Changes

- Updated dependencies
  - @tekbreed/tekmemo@1.0.0-alpha.0
